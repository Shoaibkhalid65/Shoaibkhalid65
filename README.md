## Featured Project

### 🔭 [Optical Tool](https://github.com/Shoaibkhalid65/OpticTool) — Production Android App

> A full-featured business management application delivered to a real client in the optical industry.

| | |
|---|---|
| **Type** | Freelance · Client Delivery |
| **Stack** | Kotlin · Jetpack Compose · MVVM · Room · Supabase · Dagger Hilt |
| **Status** | ✅ Production — Actively in use |

**What it does:**
- Optical prescription calculator with real-time validation (SPH, CYL, AXIS, ADD)
- Invoice generation with PDF export, WhatsApp sharing & payment tracking
- Offline-first architecture — 100% functionality without internet
- Google OAuth + Email/OTP authentication via Supabase
- Custom image compression pipeline (~250KB, 85% size reduction)
- Sales analytics dashboard and shop branding customization

**Impact:** 60% faster prescription processing · Zero billing errors · 13+ screens · 61 Kotlin files
